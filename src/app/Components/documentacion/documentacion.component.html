Documentación

Introducción

Esta documentación te ayudará a familiarizarte con los recursos de la API de Muebles y te mostrará como realizar diferentes consultas, para que puedas aprovecharlas al máximo.

REST

Base url: https://api-muebleria.onrender.com

La URL base contiene información sobre todos los recursos de la API disponibles. Todas las solicitudes son solicitudes GET y pasan por https. Todas las repuestas devolverán datos en json.

GET https://api-muebleria.onrender.com
 {{jsonString}}

Actualmente hay tres recursos disponibles

  • Muebles: usado para obtener todos los muebles
  • Marcas: usado para obtener todas las marcas
  • Proveedores usado para obtener todos los proveedores

Muebles

Existe un total de 180 muebles ordenados por id.

Esquema de muebles

Key Tipo Descripción
id string Identificador único del producto
nombre String Nombre del producto
categoria object características de la categoría del producto
marca object características de la marca del producto
costo int costo del producto
precio int precio del producto
dimensiones object características de las dimensiones del producto
color array características de los colores del producto
foto string foto del producto
fechaAdquisicion date fecha en la que fue adquirido el producto
cantidad_existente string cantidad existente del producto
status string status del producto
material_de_fabricacion array tipo de materiales del producto
origen object origen de donde proviene el producto
